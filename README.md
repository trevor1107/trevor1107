### Introducation

개발 이외의 일에서도 항상 능동적이고 적극적인 태도로 문제를 해결해 왔고, 개발자로서도 그 태도를 이어가고 있습니다.

반복되는 작업에 대해 자동화할 수 있는 방법을 고민하며, 더 나은 개발 환경을 만드는 데 관심이 있습니다.

아직은 배우는 과정에 있지만, 실질적인 도움이 되는 제품을 만들기 위해 꾸준히 성장하고 있습니다.
